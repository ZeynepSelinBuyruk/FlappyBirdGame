# Flappy Bird Game

A simple Flappy Bird clone built with HTML, CSS, and JavaScript.

## How to Play

- Press the spacebar or click the mouse to make the bird flap and avoid the pipes.
- Try to get the highest score possible!

## How to Run

1. Clone or download this repository.
2. Open `index.html` in your web browser.
3. Enjoy the game!

## Features

- Classic Flappy Bird gameplay
- Score tracking
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript
